 # 💼 Vault — Premium Skeuomorphic Financial Wallet

> **Vault** is a premium personal finance and budgeting web application designed with a realistic skeuomorphic interface. Inspired by handcrafted leather wallets and luxury desk accessories, it transforms expense tracking into an engaging and interactive financial experience through rich animations, realistic textures, and intelligent budgeting insights.

---

## 🌟 Overview

Unlike conventional budgeting applications, **Vault** recreates the feeling of managing physical finances using a handcrafted leather wallet, a metallic savings vault, and an interactive desk calendar.

The application combines modern frontend development with realistic skeuomorphic design principles to make budgeting visually appealing while remaining highly functional.

---

### 💵 Dynamic Cash Filling Meter

The wallet visually represents your financial health.

| Savings Ratio | Wallet State                |
| ------------- | --------------------------- |
| 0–20%         | Nearly Empty                |
| 20–40%        | Few Notes                   |
| 40–60%        | Moderately Filled           |
| 60–80%        | Comfortable Savings         |
| 80–100%       | Fully Packed Premium Wallet |

As your savings increase, more ₹500 currency notes become visible inside the wallet.

---

### 📅 Interactive Desk Calendar

A realistic ring-binder desk calendar allows complete monthly budgeting.

Features include:

* Previous Month navigation
* Next Month navigation
* Unlimited month history
* Future planning
* Monthly budget persistence
* Transaction history
* Daily transaction indicators

Daily status indicators:

🟢 Income

🔴 Expense

🟡 Bills & Savings

Selecting any date opens an integrated ledger drawer displaying every transaction recorded on that day.

---

### 🏦 Metallic Savings Vault

A premium brushed-steel savings safe accompanies the wallet.

Features include:

* Rotating mechanical combination dial
* Realistic vibration animations
* Opening vault door
* Gold bar stacking
* Coin accumulation
* Savings growth visualization

As savings increase, the vault gradually fills with gold bars and coins.

---

### 🤖 AI Financial Coach

Vault continuously analyzes spending patterns and automatically generates financial recommendations.

Example insights include:

* You spent 18% less this week.
* You remained within your monthly budget.
* Food spending increased by 12%.
* You saved ₹3,250 more than last month.
* Reducing restaurant expenses could save approximately ₹2,000 next month.

---

### 🌗 Light & Dark Mode

Vault supports an instant appearance switch without reloading the page.

#### ☀️ Light Theme

* Warm beige workspace
* Premium leather wallet
* Cream panels
* Metallic gold highlights

#### 🌙 Dark Theme

* Dark workspace
* Rich brown leather
* Dark interface cards
* Emerald green actions
* Premium gold highlights

All animations, shadows, textures, and 3D effects remain fully preserved in both themes.
